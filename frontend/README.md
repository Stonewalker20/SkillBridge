# Frontend

React + Vite client for SkillBridge.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run lint:fix
npm test
npm run test:watch
```

## Notes

- The workspace uses npm, backed by `package-lock.json`.
- The lint setup is flat-config based in `eslint.config.js`.
- Vitest runs in Node for the current baseline smoke test.
